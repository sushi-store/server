### Sushi store

## Project setup

1. clone repo
2. pipenv shell
3. pipenv install

## Starting migrations

4. `cd server` `python manage.py migrate`

## Project run

5. Run the server:
   `cd server` `python manage.py runserver`

## ADMIN credentials

6. To enter django administration follow the link:
   http://127.0.0.1:8000/admin and login with `admin@admin.com`:`admin`
